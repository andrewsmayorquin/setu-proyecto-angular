# Mejoras profesionales agregadas

## Incluido
- PWA con `manifest.webmanifest`, `ngsw-config.json` y `@angular/service-worker`.
- Dashboard profesional de resultados con estadísticas en tiempo real desde Firestore/localStorage.
- Gráficas accesibles con `role="img"`, `figcaption` y etiquetas ARIA.
- Animaciones suaves con Angular Animations y respeto a `prefers-reduced-motion`.
- Paginación accesible en la tabla de resultados.
- Exportación a PDF usando impresión del navegador.
- Exportación a Excel mediante CSV compatible con Excel.
- Atajos de teclado globales: Alt+H, Alt+E, Alt+R, Alt+T, Alt+U, Alt+V.
- Navegación por teclado, foco visible, skip link y nombres accesibles.
- Configuración enfocada en Lighthouse alto: lazy loading, PWA, contraste, metadatos, responsive y accesibilidad.

## Comandos
```bash
npm install --legacy-peer-deps
npm run build
npm start
```
